## 🚀 快速开始

### ⚡ 一键安装（推荐用于 VPS）

在 VPS 服务器上使用单个命令安装 iRolixBot 的最简单方法：

```bash
sudo bash <(curl -sSL https://raw.githubusercontent.com/iitzSeriZdev/iRolixBot/main/install-vps.sh)
```

**会发生什么：**
1. ✅ **自动系统更新**：脚本自动运行 `apt update && apt upgrade`
2. ✅ **交互式菜单**：系统更新后，您将看到一个菜单：
   - `1` = 安装 iRolixBot
   - `2` = 删除 iRolixBot（带备份）
   - `3` = 删除 iRolixBot（无备份 - 完全删除）
   - `4` = 更新 iRolixBot 到最新版本
3. ✅ **自动安装**：根据您的选择，脚本将：
   - 安装所有先决条件（PHP 8.2、MariaDB、Nginx、Redis、Composer）
   - 克隆存储库
   - 设置数据库和配置
   - 配置 Web 服务器（Nginx）
   - 设置 cron 作业
   - 配置权限

**替代方案：直接命令**（用于自动化）：
```bash
# 直接安装
sudo bash <(curl -sSL https://raw.githubusercontent.com/iitzSeriZdev/iRolixBot/main/install-vps.sh) install

# 带备份卸载
sudo bash <(curl -sSL https://raw.githubusercontent.com/iitzSeriZdev/iRolixBot/main/install-vps.sh) uninstall-with-backup

# 无备份卸载（完全删除）
sudo bash <(curl -sSL https://raw.githubusercontent.com/iitzSeriZdev/iRolixBot/main/install-vps.sh) uninstall-without-backup

# 直接更新
sudo bash <(curl -sSL https://raw.githubusercontent.com/iitzSeriZdev/iRolixBot/main/install-vps.sh) update
```

**安装后：**
1. 数据库凭据保存到 `/tmp/irolix_db_config`
2. 编辑 `/opt/iRolixBot/config.php` 以添加您的机器人令牌
3. 在 `http://YOUR_SERVER_IP/panel` 访问面板
4. 通过 Web 界面完成设置

### 📦 手动安装

如果您更喜欢手动安装：

#### 步骤 1：克隆存储库

```bash
git clone https://github.com/iitzSeriZdev/iRolixBot.git
cd iRolixBot
```

#### 步骤 2：安装依赖项

```bash
composer install
```

#### 步骤 3：设置数据库

创建 MySQL/MariaDB 数据库并记录凭据。

#### 步骤 4：创建 Telegram 机器人

1. 在 Telegram 上向 [@BotFather](https://t.me/BotFather) 发送消息
2. 使用 `/newbot` 创建您的机器人
3. 安全保存机器人令牌

#### 步骤 5：运行安装

1. 将文件上传到您的 Web 服务器
2. 在浏览器中访问 `install.php`
3. 按照直观的安装向导：
   - 选择您的语言
   - 输入数据库详细信息
   - 配置机器人令牌
   - 设置管理员帐户
   - 完成安装

#### 步骤 6：设置 Cron 作业

配置**仅一个 cron 作业**：

```bash
* * * * * /usr/bin/php /path/to/iRolixBot/cron/CronJob.php >> /path/to/iRolixBot/storage/logs/cron_output.log 2>&1
```

或使用 HTTP：

```bash
* * * * * curl -s https://yourdomain.com/cron/CronJob.php > /dev/null 2>&1
```

**就是这样！** 集中管理器自动处理所有事情。🎉
